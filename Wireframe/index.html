Learning Git & Web Basics

A short overview of README files, wireframes, and Git branches.

What is the purpose of a README file?

A README file introduces a project. It usually explains what the project does, how to install and use it, and may include contribution guidelines. It is often the first thing other developers see when they visit a repository.

What is the purpose of a wireframe?

A wireframe is a simple sketch that shows the structure of a web page or application. It helps designers and developers agree on layout and functionality before building the final version.

What is a branch in Git?

A branch in Git is a separate line of development. It allows you to work on features or fixes independently from the main code. Later, the branch can be merged back into the main project.
